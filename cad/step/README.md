# STEP exports

[English](#english) | [中文](#中文)

## English

This directory contains the neutral-format CAD publication set. Files are grouped by their originating fixture or specimen:

| Directory | Contents |
|---|---|
| `impact-test-rig/` | Seven impact-test-rig parts and fixture exports |
| `test-rig/` | Six general test-rig assembly and part exports |
| `vehicle-door-specimen/` | One vehicle-door specimen export |

The 14 files are byte-for-byte copies of existing local STEP exports. Source exports were not moved, rewritten, or regenerated, and no Autodesk Inventor working files are included.

Use [`SHA256SUMS.txt`](SHA256SUMS.txt) to verify file integrity. Paths and repository-authored technical documentation use English or German.

## 中文

本目录保存中性格式的 CAD 发布文件。文件按照对应夹具或试样分组：

| 目录 | 内容 |
|---|---|
| `impact-test-rig/` | 7 个冲击试验台零件或夹具导出文件 |
| `test-rig/` | 6 个通用试验台装配或零件导出文件 |
| `vehicle-door-specimen/` | 1 个车门试样导出文件 |

这 14 个文件均为现有本地 STEP 导出文件的逐字节副本。源导出文件未被移动、改写或重新生成，发布内容中也不包含任何 Autodesk Inventor 工作文件。

可使用 [`SHA256SUMS.txt`](SHA256SUMS.txt) 验证文件完整性。文件路径及仓库自行编写的技术文档使用英语或德语。
