# Flexural testing

[English](#english) | [中文](#中文)

## English

| Directory | Contents |
|---|---|
| [`analysis/`](analysis/) | Browser analysis tool and default processed CSV |
| [`docs/methods/`](docs/methods/) | Normalization methods and analysis-tool notes, organized under `en/` and `zh/` where separate language versions exist |
| [`docs/reports/`](docs/reports/) | Analysis reports with matching English and Chinese files under `en/` and `zh/` |
| [`forms/`](forms/) | Four-point test preparation and record forms plus the laboratory PDF |
| [`test-results/`](test-results/) | Original German machine-exported XLS files and matching PDF reports |

To use the analysis tool, open [`analysis/data-analysis-tool.html`](analysis/data-analysis-tool.html). Keep [`analysis/data-analysis.csv`](analysis/data-analysis.csv) beside it for default loading.

Separate translations use the same filename in sibling `en/` and `zh/` directories. The main report is available in [English](docs/reports/en/flexural-test-analysis-report.md) and [Chinese](docs/reports/zh/flexural-test-analysis-report.md). English technical documents are authoritative; Chinese versions provide reading support and preserve original project notes.

The formatted result-summary workbook remains in local archival storage until it can be converted without changing formulas or formatting.

## 中文

| 目录 | 内容 |
|---|---|
| [`analysis/`](analysis/) | 浏览器端分析工具及默认的处理后 CSV 数据 |
| [`docs/methods/`](docs/methods/) | 标准化方法及分析工具实现说明；存在独立语言版本时分别存放于 `en/` 和 `zh/` |
| [`docs/reports/`](docs/reports/) | 分别存放于 `en/` 与 `zh/` 的英语和中文分析报告 |
| [`forms/`](forms/) | 四点弯折试验准备表、记录表及实验室 PDF |
| [`test-results/`](test-results/) | 德语设备原始 XLS 导出文件及对应 PDF 报告 |

使用分析工具时，请打开 [`analysis/data-analysis-tool.html`](analysis/data-analysis-tool.html)，并将默认数据 [`analysis/data-analysis.csv`](analysis/data-analysis.csv) 保持在同一目录中。

独立翻译版本采用相同文件名，并分别放在同级 `en/` 与 `zh/` 目录中。主要分析报告提供[英语版](docs/reports/en/flexural-test-analysis-report.md)和[中文版](docs/reports/zh/flexural-test-analysis-report.md)。英语技术文档为正式版本；中文版本用于阅读支持并保存项目原始说明。

带格式的结果汇总工作簿继续保存在本地归档中，直至能够在不改变公式或格式的情况下完成转换。
