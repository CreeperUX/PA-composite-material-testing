## English

| Directory | Contents |
|---|---|
| [`analysis/`](analysis/) | Browser analysis tool and default processed CSV |
| [`docs/methods/`](docs/methods/) | Normalization methods and analysis-tool implementation notes |
| [`docs/reports/`](docs/reports/) | English analysis reports |
| [`forms/`](forms/) | Four-point test preparation and record forms plus the laboratory PDF |
| [`test-results/`](test-results/) | Original German machine-exported XLS files and matching PDF reports |

To use the analysis tool, open [`analysis/data-analysis-tool.html`](analysis/data-analysis-tool.html). Keep [`analysis/data-analysis.csv`](analysis/data-analysis.csv) beside it for default loading.

The formatted result-summary workbook remains in local archival storage until it can be converted without changing formulas or formatting.
