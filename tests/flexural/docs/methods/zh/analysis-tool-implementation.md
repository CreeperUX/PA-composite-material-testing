# 弯折测试数据分析工具工作原理

[English](../en/analysis-tool-implementation.md) | [中文](analysis-tool-implementation.md)

本文档根据 [`data-analysis-tool.html`](../../../analysis/data-analysis-tool.html) 的界面结构和内嵌 JavaScript 逻辑整理，用于说明该工具如何读取弯折测试数据、进行筛选与标准化、识别屈服特征、生成统计图并导出结果。

## 1. 工具定位

`data-analysis-tool.html` 是一个单文件浏览器端分析工具，主要处理三点弯折测试结果 CSV。它不依赖后端服务器完成计算，核心逻辑全部在浏览器中运行。

主要功能包括：

- 读取默认 `data-analysis.csv` 或手动选择 CSV 文件。
- 按试件类型、试件编号、测试批次、厚度范围和采样序号筛选曲线。
- 在原始表观应力、厚度标准化、D7249 简化表层应力、面密度比性能之间切换。
- 自动识别每个试件的峰值点、屈服点、初始斜率和曲线面积。
- 生成单条曲线、材料类型均值曲线、线性段裁切均值曲线和论文统计图。
- 导出论文统计图为 PNG、SVG 或 CSV。

## 2. 数据输入与校验

### 2.1 数据来源

工具提供两种读取方式：

1. 点击“读取 data-analysis.csv”，通过 `fetch("data-analysis.csv")` 从 HTML 同目录读取默认文件。
2. 通过文件选择框手动载入 CSV。

如果浏览器安全策略阻止本地 `fetch`，默认读取可能失败，此时应使用文件选择框，或通过本地 HTTP 服务打开 HTML。

### 2.2 必需列

CSV 必须包含以下列名，列名不匹配会直接报错：

| 列名 | 用途 |
| --- | --- |
| `测试编号` | 提取测试批次，例如 `01_24` 或 `01_24_H` |
| `试件类型` | 材料类型，目前主要识别 `3k`、`1x1`、`2x2` |
| `统一试件编号` | 同一类型内部的统一编号，用作曲线和屈服图横坐标 |
| `原始Probe编号` | 显示在系列列表中，便于追溯原始试件 |
| `源文件` | 记录数据来源文件 |
| `厚度 h [mm]` | 标准化和厚度筛选输入 |
| `采样序号` | 曲线点排序和采样区间筛选 |
| `Verformung [mm]` | 变形/位移，作为曲线横坐标 |
| `Standardkraft [MPa]` | 表观应力，作为原始曲线纵坐标 |

### 2.3 CSV 解析

工具自带 `parseCsv()`，支持：

- UTF-8 BOM 去除。
- 逗号分隔。
- 双引号包裹字段。
- 字段内双引号转义。
- 忽略空行。

解析后，`loadCsvText()` 会把关键数值列转换为 `Number`，并过滤掉没有有效 `统一试件编号`、`Verformung [mm]` 或 `Standardkraft [MPa]` 的行。

## 3. 内部数据模型

### 3.1 行数据

CSV 的每一行被转换为一个采样点对象：

```js
{
  testId,
  type,
  unifiedId,
  originalProbe,
  sourceFile,
  thickness,
  sampleIndex,
  deformation,
  stress
}
```

其中 `deformation` 来自 `Verformung [mm]`，`stress` 来自 `Standardkraft [MPa]`。

### 3.2 试件分组

`buildSpecimens()` 按以下键合并采样点：

```text
试件键 = 试件类型 + "|" + 统一试件编号
```

每个试件对象包含：

- `type`：材料类型。
- `unifiedId`：统一试件编号。
- `name`：显示名称，例如 `Roving-03`。
- `batch`：从 `测试编号` 中提取的测试批次。
- `thickness`：该试件厚度。
- `points`：按 `采样序号` 升序排列的曲线点。

材料类型显示名称固定为：

| 原始类型 | 显示名称 |
| --- | --- |
| `3k` | Roving |
| `1x1` | Plain |
| `2x2` | Twill |

## 4. 筛选逻辑

所有图表和统计结果都基于当前可见试件。`getVisibleSpecimens()` 依次应用以下筛选条件：

1. 材料类型复选框：`3k`、`1x1`、`2x2`。
2. 系列选择列表：用户勾选的具体试件。
3. 测试批次：从 `测试编号` 中识别，例如 `01_21`、`01_24_H`。
4. 统一试件编号范围。
5. 厚度范围。

采样序号范围不删除试件本身，只在 `getDisplayPoints()` 中过滤每条曲线的点。

## 5. 图表模式

工具有四种主视图：

| 模式 | 计算方式 |
| --- | --- |
| 原始 | 直接显示 `Standardkraft [MPa]` 对 `Verformung [mm]` |
| 标准化 | 对每个点调用 `normalizeStress()` 后显示 |
| 类型均值 | 按材料类型重采样并求平均曲线 |
| 类型均值：带裁切 | 先自动裁切每条曲线的线性段，再按类型求平均 |

在原始与标准化视图下，曲线图会标出每个试件的屈服点和峰值点。均值视图只显示均值曲线。

## 6. 标准化计算

标准化只影响“标准化”及均值相关视图。原始视图始终使用原始 `Standardkraft [MPa]`。

### 6.1 不进行标准化

模式为 `none` 时：

$$
\sigma_{\mathrm{display}} = \sigma_{\mathrm{app}}
$$

此模式下材料类型修正系数不生效。

### 6.2 截面惯性矩参考 I0/I

该模式把曲线换算到参考厚度 `h0`。代码中假设试件宽度相同，使用整矩形截面的惯性矩比例：

$$
I \propto h^3
$$

实际计算为：

$$
\sigma_{\mathrm{norm}}
=
\sigma_{\mathrm{app}}
\frac{h_0^3}{h^3}
=
\sigma_{\mathrm{app}}
\left(\frac{h_0}{h}\right)^3
$$

其中：

- `h` 为当前试件厚度，来自 CSV 的 `厚度 h [mm]`。
- `h0` 为界面输入的参考厚度，默认 `2.00 mm`。

若厚度无效或参考厚度小于等于 0，工具会保留原始值。

### 6.3 D7249 简化表层应力

该模式把实心梁表观应力近似换算为夹芯结构表层应力。工具假设：

- 三点弯曲。
- 上下表层对称。
- 失效主要由表层主导。
- CSV 中没有宽度列时，不显式使用宽度。

总厚度记为 `d`，单侧表层厚度记为 `t`，芯层厚度为：

$$
c = \max(d - 2t, 0)
$$

代码中还会限制：

$$
t \le 0.98 \times \frac{d}{2}
$$

表观应力到表层应力的换算为：

$$
\sigma_f
=
\sigma_{\mathrm{app}}
\frac{d^2}{3t(d+c)}
$$

单侧表层厚度 `t` 按材料类型读取。当前试件为两侧各 4 层纤维的三明治结构；Plain/Twill 默认值来自材料报告面密度和固化体积分数估算，Roving 默认值进一步受当前实测总厚度约束：

| 材料类型 | 工具显示 | 默认单侧表层厚度 `t` |
| --- | --- | ---: |
| `3k` | Roving | `0.780 mm/侧` |
| `1x1` | Plain | `0.576 mm/侧` |
| `2x2` | Twill | `0.568 mm/侧` |

界面允许直接修改三种材料的 `t`，用于敏感性分析或替换为后续显微截面、铺层实测值。D7249 标准化计算会把当前试件的 `type` 传入 `getFaceThickness(type)`，因此单条曲线、类型均值、线性裁切均值、屈服/峰值统计，以及面密度比性能中选择“D7249 简化表层应力”作为预先标准化时，都会使用对应材料自己的表层厚度。

### 6.4 基于面密度的比性能

该模式先执行一个“预先标准化”步骤，再除以材料类型对应的面密度：

$$
y_{\mathrm{specific}}
=
\frac{y_{\mathrm{base}} \cdot k_{\mathrm{type}}}{A_{\mathrm{type}}}
$$

其中：

- `y_base` 来自预先标准化，可选“不预先标准化”“截面惯性矩参考 I0/I”或“D7249 简化表层应力”。
- `k_type` 为材料类型修正系数。
- `A_type` 为对应材料类型的面密度，默认三类均为 `132.9 g/m²`。

界面显示单位为：

```text
MPa·m²/g
```

若面密度无效或小于等于 0，工具只返回 `y_base * k_type`。

### 6.5 材料类型修正系数

当标准化模式不是 `none` 时，会读取每种材料类型的人工修正系数：

| 类型 | 输入框 |
| --- | --- |
| Roving / `3k` | `factor3k` |
| Plain / `1x1` | `factor1x1` |
| Twill / `2x2` | `factor2x2` |

普通标准化模式中：

$$
y_{\mathrm{display}}
=
y_{\mathrm{mode}} \cdot k_{\mathrm{type}}
$$

面密度比性能模式中，系数在除以面密度前应用。

## 7. 屈服点、峰值与曲线面积

每个可见试件都会通过 `computeFeatures()` 提取特征。特征提取基于当前显示模式，因此原始视图和标准化视图得到的数值可能不同。

### 7.1 峰值点

峰值点是当前显示曲线中纵坐标最大的点：

$$
P_{\mathrm{peak}}
=
\arg\max(y)
$$

这里的 `y` 可以是原始表观应力、标准化应力、表层应力或面密度比性能。

### 7.2 初始斜率

工具取曲线前一段作为弹性段，默认比例为 `0.18`，可在界面中修改。该比例会被限制在 `0.05-0.45`。

弹性段终点索引为：

$$
n_{\mathrm{elastic}}
=
\max(5,\ \lfloor n \cdot f_{\mathrm{elastic}} \rfloor)
$$

初始斜率通过最小二乘直线拟合得到：

$$
m
=
\frac{n\sum xy-\sum x\sum y}
{n\sum x^2-(\sum x)^2}
$$

### 7.3 屈服点识别

工具先对纵坐标做窗口大小为 5 的移动平均。随后从弹性段附近开始，计算平滑曲线的局部斜率：

$$
m_i
=
\frac{y_{i+2}-y_{i-2}}{x_{i+2}-x_{i-2}}
$$

当同时满足以下条件时，第一个满足条件的点被认为是屈服点：

$$
y_i > 0.18 \cdot y_{\mathrm{peak}}
$$

$$
m_i < m_{\mathrm{initial}} \cdot r_{\mathrm{drop}}
$$

其中 `r_drop` 是“斜率衰减阈值”，默认 `0.75`，允许范围为 `0.20-0.95`。

如果没有找到明显斜率衰减点，则把峰值点作为屈服特征点。

### 7.4 曲线面积

曲线面积通过梯形积分计算：

$$
A
=
\sum_{i=1}^{n-1}
\frac{(x_i-x_{i-1})(y_i+y_{i-1})}{2}
$$

在应力-变形曲线中，该值可作为相对吸能或曲线综合响应指标，但它不是严格能量，除非纵坐标与横坐标单位经过相应物理换算。

## 8. 均值曲线与参考线

### 8.1 类型均值曲线

`makeAveragedSeries()` 按材料类型分组，对每组曲线进行线性插值重采样。默认重采样点数为 `180`，用户可设置，范围限制为 `50-500`。

对每个材料类型：

1. 找到该类型所有曲线的最大终点变形。
2. 在 `0` 到最大终点变形之间建立等间距横坐标。
3. 对每条曲线在该横坐标处插值。
4. 对有效插值结果求平均。

### 8.2 类型均值：带裁切

`makeLinearAveragedSeries()` 会先用 `cropToLinearSegment()` 裁切每条曲线。裁切逻辑与屈服识别相近：

- 使用初始弹性段斜率作为基准。
- 使用移动平均曲线计算局部斜率。
- 当局部斜率低于 `初始斜率 × 斜率衰减阈值` 时停止。

带裁切均值曲线只在所有裁切后曲线共同覆盖的变形范围内求平均，因此横坐标终点取各条裁切曲线终点的最小值。

### 8.3 参考线

参考线用于在单条曲线图上观察整体趋势，支持：

| 选项 | 含义 |
| --- | --- |
| 不显示 | 不绘制参考线 |
| 各类型均值线 | 每种材料类型一条虚线均值曲线 |
| 全部选中均值线 | 对所有选中试件求一条总均值线 |
| 各类型 + 全部均值线 | 同时显示两类参考线 |

在线性段裁切均值模式下，参考线会被隐藏。

## 9. 统计输出

### 9.1 顶部 KPI

顶部指标基于当前筛选结果实时计算：

- 试件数量。
- 当前采样点总数。
- 各材料类型试件数。
- 平均峰值。
- 平均屈服点。

### 9.2 材料类型对比表

`renderSummaryTable()` 按材料类型汇总：

| 指标 | 计算方式 |
| --- | --- |
| `n` | 当前筛选下该类型试件数量 |
| 平均厚度 | 试件厚度均值 |
| 初始斜率 | 各试件初始斜率均值 |
| 屈服应力 | 各试件屈服点纵坐标均值 |
| 屈服变形 | 各试件屈服点横坐标均值 |
| 峰值应力 | 各试件峰值点纵坐标均值 |
| 峰值变形 | 各试件峰值点横坐标均值 |
| 曲线面积 | 各试件梯形积分面积均值 |

### 9.3 论文统计图

论文统计图按材料类型绘制柱状图，并叠加单个试件数据点。可选指标包括：

- 峰值应力。
- 屈服应力。
- 初始斜率。
- 曲线面积。
- 厚度。

误差条可选：

| 选项 | 计算公式 |
| --- | --- |
| SD | 样本标准差，方差分母为 `n - 1` |
| SEM | `SD / sqrt(n)` |
| 95% CI | `1.96 × SEM` |
| 无误差条 | 误差为 0 |

统计值由 `makePaperStats()` 生成，只保留有有效数值的材料类型。

## 10. 导出逻辑

论文统计图支持三种导出格式：

| 格式 | 文件名格式 | 内容 |
| --- | --- | --- |
| PNG | `flexural_figure_<metric>_600dpi.png` | 780 × 540 逻辑尺寸，5 倍缩放绘制 |
| SVG | `flexural_figure_<metric>.svg` | 矢量柱状图 |
| CSV | `flexural_statistics_<metric>.csv` | 分组统计表 |

CSV 导出列为：

```text
type,n,mean,sd,sem,ci95,error_mode,metric
```

数值最多保留 6 位小数。

## 11. 交互与刷新机制

工具采用事件驱动刷新：

- 切换语言会重新应用所有界面文本并刷新图表。
- 修改筛选条件会重置曲线缩放视图并重新计算。
- 修改标准化参数、材料修正系数或屈服识别参数会重算全部特征。
- 曲线图支持鼠标滚轮缩放、拖拽平移和重置视图。
- 浏览器窗口尺寸变化时会重新绘制 Canvas。

所有图表使用 HTML Canvas 直接绘制，没有使用外部绘图库。

## 12. 重要假设与限制

1. `试件类型 + 统一试件编号` 必须能唯一代表一个试件。如果不同测试批次中重复了相同组合，工具会把它们合并到同一试件。
2. 曲线点按 `采样序号` 排序，插值均值曲线默认 `Verformung [mm]` 随采样序号总体单调增加。
3. 截面惯性矩参考模式假设试件宽度一致，因为 CSV 中没有宽度列。
4. D7249 简化表层应力模式是由表观应力换算得到的近似值，适用于对称夹芯、三点弯和表层主导失效情形。
5. 屈服点是算法识别点，不是人工判读点；它会随采样范围、平滑窗口、初始弹性段比例和斜率衰减阈值变化。
6. 面密度比性能依赖用户输入的面密度和材料类型修正系数，默认值相同不代表三类材料真实面密度完全一致。
7. PNG 导出通过高分辨率 Canvas 绘制实现，文件名标注 `600dpi`，但代码没有写入独立 DPI 元数据。

## 13. 推荐使用流程

1. 确认 `data-analysis.csv` 与 HTML 在同一目录，或手动选择正确 CSV。
2. 检查试件类型、批次、编号和厚度是否识别正确。
3. 根据比较目的选择视图：
   - 只看原始测试结果：使用“原始”。
   - 比较厚度修正后结果：使用“标准化”并选择合适标准化方式。
   - 看材料类型整体趋势：使用“类型均值”。
   - 只比较初始线性响应：使用“类型均值：带裁切”。
4. 根据材料结构选择标准化方式：
   - 近似实心矩形截面：优先使用截面惯性矩参考。
   - 夹芯或表层主导结构：使用 D7249 简化表层应力，并检查三种材料的表层厚度设置。
   - 比较单位面密度效率：使用面密度比性能。
5. 调整屈服识别参数，确认屈服点标记与曲线形态一致。
6. 使用材料类型对比表核对均值，再导出论文统计图或 CSV。
