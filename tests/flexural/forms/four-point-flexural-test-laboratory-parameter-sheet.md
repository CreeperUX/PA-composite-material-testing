| Item                                              | Setting | Unit | Verified / measured value |

| Four-point fixture ID                             |         |    - |                           |

`*` The historical three-point report used a 2.25 mm radius; verify the actual four-point fixture.

The loading-head speed of `2.50 mm/min` was converted for the four-point geometry (`S=60 mm`, `L=20 mm`) to maintain the same nominal outer-fibre strain rate as the historical three-point test (`S=40 mm`, `1.00 mm/min`).
